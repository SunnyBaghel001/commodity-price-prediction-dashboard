# 📊 Commodity Price Prediction Dashboard

An end-to-end Machine Learning application that predicts commodity prices and provides interactive market analytics using Streamlit.

---

## 🚀 Live Demo

🔗 https://commodity-price-prediction-dashboard-3001.streamlit.app/

---

## 📌 Project Overview

This project uses historical APMC (Agricultural Produce Market Committee) data to predict the **modal price** of commodities based on factors like:

- Commodity type
- State
- Month & Day
- Price spread (Min–Max)
- Historical trends

The model is deployed as an **interactive dashboard** using Streamlit.

---

## 🧠 Machine Learning

### Models Used:
- Random Forest Regressor
- XGBoost Regressor

### Performance:
- 📈 R² Score: ~0.76
- 📉 MAE: ~590
- 📊 RMSE: ~830

---

## ⚙️ Features

- 🔮 Predict commodity modal price
- 📦 Shows price in:
  - ₹ per quintal
  - ₹ per kg
- 📊 Interactive charts:
  - Monthly price trends
  - State-wise comparison
  - Commodity ranking
- 🔁 Dynamic filtering based on user input
- ⚡ Real-time predictions
